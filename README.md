# CLI To-Do App (Python)

A clean and modular command-line To-Do application built using Python.

## Features
- Add tasks with due date and priority
- View tasks (pending first, completed at bottom)
- Mark tasks as done
- Edit tasks
- Delete tasks
- Search and filter tasks
- Persistent storage using JSON

## Tech Stack
- Python 3
- JSON for persistence

## How to Run

python3 todo1.py
